# 🔥 Price Tracker — مراقب الأسعار

> أداة سطح مكتب مبنية بـ Python لمراقبة أسعار المنتجات تلقائياً وإشعارك فور نزول السعر.

---

## 📸 نظرة عامة

**Price Tracker** هي أداة مفتوحة المصدر تتيح لك إضافة روابط منتجات من مواقع التسوق (مثل Amazon)، ومراقبة أسعارها بشكل تلقائي على فترات زمنية محددة، مع إرسال إشعار فوري على سطح المكتب عند انخفاض السعر.

---

## ✨ المميزات

| الميزة | التفاصيل |
|--------|----------|
| 🛒 إضافة منتجات | أضف اسم المنتج ورابطه بسهولة |
| 💰 جلب السعر تلقائياً | يجيب السعر الحالي فور الإضافة |
| 🔄 مراقبة دورية | فحص تلقائي كل 30 دقيقة أو ساعة أو 3/6 ساعات |
| 🔔 إشعارات سطح المكتب | تنبيه فوري لما السعر يقل عن الأصلي |
| 📊 متابعة التغيير | يعرض الفرق بين سعر البداية والسعر الحالي |
| 🖥️ سجل مراقبة | لكل منتج سجل كامل بكل عمليات الفحص |
| 🗑️ حذف المنتجات | إزالة أي منتج من القائمة بضغطة زر |
| 🌙 واجهة داكنة | تصميم عصري Dark Mode مريح للعين |

## 🛠️ المكتبات المطلوبة

```bash
pip install customtkinter requests beautifulsoup4 plyer
```

| المكتبة | الاستخدام |
|---------|----------|
| `customtkinter` | بناء واجهة المستخدم الرسومية بتصميم عصري |
| `requests` | إرسال طلبات HTTP لجلب صفحات المنتجات |
| `beautifulsoup4` | تحليل HTML واستخراج بيانات السعر |
| `plyer` | إرسال إشعارات على سطح مكتب Windows/macOS/Linux |
| `threading` | تشغيل عمليات الفحص في الخلفية بدون تجميد الواجهة |
| `json` | حفظ وتحميل بيانات المنتجات من ملف محلي |

> المكتبات `threading` و `json` و `os` و `datetime` و `time` جزء من Python القياسية ولا تحتاج تثبيت.

## 📖 طريقة الاستخدام

### ➕ إضافة منتج جديد
1. اكتب **اسم المنتج** في الخانة الأولى
2. الصق **رابط المنتج** من الموقع في الخانة التانية
3. اضغط **"أضف المنتج وراقب"**
4. الأداة هتجيب السعر الحالي وتبدأ المراقبة تلقائياً

### 🔍 فحص يدوي فوري
- اضغط **"فحص دلوقتي"** لفحص كل المنتجات فوراً بدون انتظار

### 📋 عرض تفاصيل منتج
- اضغط على أي منتج في القائمة لفتح نافذة التفاصيل التي تعرض:
  - سعر البداية والسعر الحالي والفرق بينهم
  - ضبط فترة الفحص التلقائي
  - سجل كامل بكل عمليات المراقبة مع الوقت

### ⏱️ ضبط فترة الفحص
- من نافذة تفاصيل المنتج، اختار الفترة المناسبة:
  - **30 دقيقة** — للمتابعة الدقيقة
  - **ساعة** — الافتراضي
  - **3 ساعات**
  - **6 ساعات**

## 🗂️ بنية المشروع

```
price_tracker/
│
├── price_tracker.py        # الملف الرئيسي للأداة
└── products.json           # قاعدة البيانات المحلية (يتولد تلقائياً)
```

### تفاصيل `products.json`
الأداة بتحفظ بيانات كل منتج تلقائياً بالشكل ده:

```json
[
  {
    "name": "لاب توب لينوفو",
    "url": "https://www.amazon.eg/...",
    "initial_price": 25000,
    "current_price": 23500,
    "status": "🔴 نزل!",
    "interval": 60,
    "logs": [
      "[14:32:01] 🔍 جاري جلب السعر الأولي...",
      "[14:32:03] ✅ السعر: 25,000",
      "[15:32:03] 🔄 فحص تلقائي...",
      "[15:32:05] 🔥 نزل من 25,000 لـ 23,500!"
    ]
  }
]
```

## 🌐 المواقع المدعومة

الأداة حالياً مهيأة لاستخراج الأسعار من **Amazon** (بما فيها Amazon Egypt) عبر CSS selectors المخصصة:

- `span.a-price-whole`
- `span#priceblock_ourprice`
- `span#priceblock_dealprice`
- `span.a-offscreen`

> **ملاحظة:** بعض الصفحات قد تستخدم JavaScript لتحميل الأسعار أو تضع حماية ضد السكرابينج، مما قد يمنع استخراج السعر في بعض الأحيان.
